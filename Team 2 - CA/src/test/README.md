# Tests

JUnit 5 tests for IPOS-CA services.

| File | What it tests |
|------|--------------|
| StockServiceTest.java | Stock management |
| SaleServiceTest.java | Sales processing |
| AccountServiceTest.java | Account holders |
| WholesaleOrderServiceTest.java | Wholesale orders |

To run: right-click any test file in IntelliJ → Run. Add JUnit 5 to classpath if prompted.
